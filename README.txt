===================
SunSpot Module
===================

**A robust non-parametric quality control procedure, based on the block boostrap and the CUSUM chart, to monitor a panel of non-Normal observations prone to noise and missing values. The method also provides predictions of the forms and the sizes of the deviations using support vector machine procedures**

The module contains different files to preprocess, train and monitor a panel of autocorrelated series with potential missing values. 
The methodology, though generic, is also applied on the number of sunspots (Ns) and the number of sunspot groups (Ng) as examples.
