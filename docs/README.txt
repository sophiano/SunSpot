===================
docs
===================

This folder contains jupyter notebooks.
Those can be open online at the following link: https://jupyter.org/try
In these notebooks, the monitoring method is applied to the long-term errors of the composite 'Nc'. 
Files with '365' in their extension indicate that the method is applied to the data smoothed on 365 days. Whereas files with '27' in their extension indicate that the method is applied to the data smoothed on 27 days. 
The notebooks can be used to reproduce the main figures of the article.
